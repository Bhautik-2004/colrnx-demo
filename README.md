# Colrnx Demo

This is a Next.js project showcasing various UI components and demo sections.

## Prerequisites

Before you begin, ensure you have met the following requirements:
* Node.js installed on your machine
* A package manager like npm, yarn, pnpm, or bun

## Installation

To install the project dependencies, run the following command:

```bash
npm install
```

## Running the Application

To start the development server, run:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

* `app/`: Contains the Next.js App Router pages and layouts.
* `components/`: Contains reusable generic React components and specialized demo features.
* `lib/`: Contains utility functions.
* `public/`: Contains static assets like images and icons.

## Tech Stack

* Next.js
* React
* Tailwind CSS
* Radix UI Form Primitives (via Shadcn UI structure)
